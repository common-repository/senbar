=== Senbar ===

Requires at least: 6.0
Tested up to: 6.0.1
Stable tag: 1.0
License: GPLv2 or later

== Description ==
Senbar hides the admin toolbar in front end for specific user roles.

== Installation ==

Upload the plugin to the wordpress plugin directory and activate it.
Go to settings->senbar config and add the specific user roles to hide the admin toolbar in front end.
